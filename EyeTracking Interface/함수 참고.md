**PyWin32 마우스 컨트롤하기**

PyWin32를 이용해서 마우스 커서의 위치를 찾거나 이동시키고, 클릭, 드래그 등의 동작을 간단하게 수행할 수 있습니다.

## SetCursorPos()

```python
import win32api

pos = (200, 200)
win32api.SetCursorPos(pos) # win32api.SetCursorPos((x, y))
```

SetCursorPos() 메서드는 마우스 커서의 위치를 입력받은 튜플의 위치((x, y))로 이동시킵니다.
