## OpenGL

​    

**OpenGL의 정의**

■ 2 3 차원 또는 차원 드로잉을 위한 표준 그래픽스 라이브러리

- 그래픽스 하드웨어에 대한 소프트웨어 인터페이스
- C나 C++과 같은 프로그래밍 언어는 아님
- 그래픽스 하드웨어에 잘 구현될 수 있음
- C 언어 기반 라이브러리
- 상태 기반 아키텍쳐
- (Immediate mode) 즉시 모드 기반

​

- GL(Graphics Library)
